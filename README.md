# Sonar - the opensource AI Search augementation tool
Sonar is an opensource AI-powered search augementation tool that enhances search capabilities by integrating the power of AI and Brave Search. It allows to perform advance searches using natural. Check it out at [sonar.ultimatedev.space](https://sonar.ultimatedev.space).

## Features
- AI-powered search augmentation: Get more relevant search results throuhgh the Brave Search API.

- Multi Model Support: choose from a variety of AI models to tailor your search experience.

- User-friendly Interface: Easy to use interface for seamless search experience.

- Open Source: Fully open source, allowing for customization and community contributions.

## Installation
1. Clone the repository:
    ```bash
    git clone https://github.com/prathamghaywat/sonar.git
    cd sonar
    ```
2. Install the required dependencies:
    ```bash
    npm install
    ```
3. Set up your environment variables:
    -  Rename `.env-example` to `.env.local` and add your Brave Search API key and preferred AI Model Base URL and API key.
4. Start the development server and access the application at `http://localhost:3000`:
    ```bash
    npm run dev
    ```
Alternatively, you can deploy Sonar on platforms like Vercel or Netlify for easy hosting. I recommend using Vercel for seamless integration with Next.js applications. But for self-hosting, follow you might want to use coolify, since it is the easiest to setup and use.
